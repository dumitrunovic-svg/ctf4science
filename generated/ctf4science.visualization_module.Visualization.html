ctf4science.visualization_module.Visualization#

class ctf4science.visualization_module.Visualization(config_path: Path | str | None = None)#

Bases: object

Generates visualizations of model predictions and evaluation metrics.

Configuration is loaded from a YAML file (default or custom path) and stored in config.

Parameters:
config_pathstr, optional

Path to a custom YAML config file. If None, uses default from package.

Methods

compare_prediction(truth, predictions[, ...])

Create a side-by-side comparison of test data and predicted data.

generate_all_plots(dataset_name, batch_path, ...)

Generate all applicable plots for a dataset based on its configuration.

plot_errors(errors[, labels])

Plot error metrics over time or across sub-datasets.

plot_from_batch(dataset_name, pair_id, batch_id)

Plot data from a run directory, supporting multiple plot types.

plot_histograms(truth, predictions, modes, bins)

Plot histograms of variables for dynamical systems over the last 'modes' time steps.

plot_prediction(ax, data[, vmin, vmax, ...])

Plot a 2D prediction on the given axes with customizable display options.

plot_psd(truth, predictions, k, modes[, labels])

Plot Power Spectral Density for spatio-temporal systems over the last k time steps.

plot_trajectories(truth, predictions[, labels])

Plot stacked trajectories for each variable, comparing truth and predictions.

save_figure_results(fig, dataset_name, ...)

Save the figure to the results directory under a visualizations subfolder.
Raises:
FileNotFoundError

If the custom config file does not exist.

Notes

plot_trajectories

  • truth : (time_steps, variables). predictions : list of same shape. labels : optional.

  • Returns plt.Figure. One subplot per variable; raises ValueError if shapes mismatch.

plot_errors

  • errors : Dict[str, List[float]]. labels : optional. Returns plt.Figure.

plot_histograms

  • truth, predictions, modes, bins. Last modes steps per variable. Returns plt.Figure.

plot_psd

  • truth, predictions, k, modes. Log PSD over last k steps. Returns plt.Figure.

plot_from_batch

  • dataset_name, pair_id, batch_id, plot_type (trajectories, histograms, psd, errors, 2d_comparison).

  • Loads data from batch dir; returns plt.Figure.

generate_all_plots

  • dataset_name, batch_path. Generates all applicable plot types per pair dir. No return value.

save_figure_results

  • fig, dataset_name, model_name, batch_name, pair_id, plot_type, optional results_dir.

  • Saves figure to results/…/visualizations/{plot_type}.png. No return value.

plot_prediction

  • ax, data (2D), optional vmin/vmax, show_ticks, show_xlabel, show_ylabel.

  • Returns AxesImage.

compare_prediction

  • truth, predictions, optional cbar_options, show_ticks, show_titles.

  • Side-by-side truth, prediction, error. Returns plt.Figure.
