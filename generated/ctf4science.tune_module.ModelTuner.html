ctf4science.tune_module.ModelTuner#

class ctf4science.tune_module.ModelTuner(config_path: str, model_name: str | None = None, save_final_config: bool = True, metric: str = 'score', mode: str = 'max', ignore_reinit_error: bool = False, time_budget_hours: float = 24.0, use_asha: bool = False, asha_config: Dict[str, Any] | None = None, gpus_per_trial: int = 0, enable_performance_monitoring: bool = False, performance_output_dir: str | None = None, ray_results_dir: str | None = None, run_opt_main: Callable[[str], None] | None = None)#

Bases: object

Orchestrates hyperparameter tuning for CTF models using Ray Tune.

Supports tuning with a specific config file or automatically detecting all config files in a model’s tuning_config/config_*.yaml.

Parameters:
config_pathstr

Path to the configuration file (dataset, model, hyperparameters).

model_namestr, optional

Model name; if not provided, inferred from config or directory.

save_final_configbool, optional

Whether to save the final optimal config, by default True.

metricstr, optional

Metric to optimize, by default "score".

modestr, optional

Optimization mode "min" or "max", by default "max".

ignore_reinit_errorbool, optional

Whether to ignore Ray reinit errors (dev only), by default False.

time_budget_hoursfloat, optional

Max time budget in hours, by default 24.0.

use_ashabool, optional

Whether to use ASHA scheduler, by default False.

asha_configdict, optional

ASHA config (max_t, grace_period, reduction_factor, brackets).

gpus_per_trialint, optional

GPUs per trial (0 = all available), by default 0.

enable_performance_monitoringbool, optional

Enable performance (time) monitoring, by default False.

performance_output_dirstr, optional

Directory for performance results.

ray_results_dirstr, optional

Directory for Ray temporary results.

run_opt_mainCallable[[str], None], optional

Callable that runs a single optimization trial given a config path (typically run_opt.main from the model directory).

Methods

run_from_cli([description])

This method provides a simple interface for running tuning from command line.

run_optimization()

Run the complete optimization workflow.
Raises:
ValueError

If config is missing required sections (dataset, model, hyperparameters), or if run_opt_main is not provided.

Notes

_construct_output_dir

  • Returns Path: results/tune_results/{model_name}/{dataset_name}/pair_id_{pair_ids}/{timestamp}/.

_infer_model_name (static)

  • config_path : str. Path to the config file.

  • Returns str: model name inferred from path (e.g. parent of tuning_config).

_validate_config

  • config : Dict. Configuration dictionary.

  • Raises ValueError if required sections missing. No return value.

_validate_param_space

  • param_space : Dict. Hyperparameter space.

  • Raises ValueError if empty or invalid types/bounds. No return value.

_objective

  • config : Dict. Trial hyperparameters.

  • Uses the injected run_opt_main callable, then sums results for pair_ids. Returns Dict[str, float] with the optimization metric.

_parse_pair_ids (static)

  • dataset_config : Dict. Dataset section of config.

  • Returns Optional[List[int]]: pair_ids or None for ‘all’.

_sum_results

  • results : dict. Results from run_opt (pairs with metrics).

  • Returns float: sum of all metric values.

_create_search_space

  • tuning_config : Dict. Hyperparameter spec (type, lower_bound, upper_bound, etc.).

  • Returns Dict[str, Any]: Ray Tune search space.

_generate_config

  • config : Dict. Selected hyperparameters. template : Dict. Config template. name : str. Output filename base.

  • Writes YAML to output_dir. Returns str: path to generated file.

_get_resources

  • Returns Dict[str, Any]: cpu and gpu resource config for Ray Tune.

run_optimization

  • Runs full tuning: creates search space, Tuner, fit(). Saves best config and history when save_final_config is True. No return value.

run_from_cli (static)

  • description : str. Parser description.

  • Parses CLI, imports the model-local run_opt.main, and runs ModelTuner for each config. No return value.
