# Ocean DAS Dataset

A spatio-temporal dataset using a novel geophysical sensing technology called DAS (Distributed Acoustic Sensing) [1]. This dataset is comprised of data from a shallow offshore DAS about 30m below sea level where surface gravity waves are particularly dispersive. The data is sampled at 5Hz but low-pass filtered to 1Hz.

[1] Yin, Jiuxun, et al. "Real‐data testing of distributed acoustic sensing for offshore earthquake early warning." The Seismic Record 3.4 (2023): 269-277.
